OpenNPO: APIs For Nonprofits

OpenNPO is an project that provides developer access to available nonprofit information through an application programming interface (API). The intent of the project to build applications that benefit nonprofits.

Nonprofits

Wikipedia defines an API as: "An application programming interface (API) is the interface that a computer system, library, or application provides in order to allow requests for services to be made of it by other computer programs, and/or allow data to be exchanged between them." OpenNPO is created in the belief that APIs can help nonprofits better manage their data and provide a greater social impact. Many nonprofits generate data that can be useful to the general public or other nonprofit organizations. Opportunities and social impact is increased with the availability of public data through technology platforms. APIS are essential for nonprofits who are looking to innovate in their causes. For more information, please read the Nonprofit Technology Network Report, "How Open APIs Can Change How Nonprofits Manage Data".

Pub 78 API Documentation

IRS Publication 78

Publication 78 is a list of organizations eligible to receive tax-deductible charitable organizations. Publication 78 is available as a text file from the IRS website found here. Or you can manually search the list on the IRS site here. The Publication 78 is updated on a monthly basis. OpenNPO has created an REST api for access to Publication 78 which returns the results in a JSON format. The api is hosted on Github with documentation for use.

GET search

Returns available nonprofit information that match a specified query.

Resource URL
http://opennpo.info/search.json

Parameters

A minimum of one parameter is required. Parameters should be URL encoded.

dba
Doing Business As. Returns nonprofit organizations with a name similar to the given parameter value.
ein
Employer ID Number. Returns nonprofit organization that match the given parameter value.
city
Returns nonprofit organizations that are registered in the city parameter value.
state
Returns nonprofit organizations that are registered in the state parameter value. Parameter value must match the two letter state code.
page
Must be used with other query parameters. Returns the paged nonprofit organizations. Defaults to first page if not included. Each request returns a max of 50 results per page.
Example Request
GET http://opennpo.info/search.json?city=San+Francisco&state=CA
Returns all nonprofit organizations registered in San Francisco, CA
